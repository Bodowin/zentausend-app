# Paket D – Scope

## Enthalten

- Stabile Identität für Stammspieler und wiederkehrende Gäste.
- IDs in neuen Spieler- und Zugdatensätzen.
- Alias-Verknüpfung bei ausdrücklicher Kader-Umbenennung.
- Identitätsbasierte Bestenliste, Form, Duelle, Angstgegner und Awards.
- Rückwärtskompatibilität für alle bisherigen Spiele und Backups.

## Nicht enthalten

- Automatische Zusammenführung ähnlich klingender Namen.
- Zentrale Cloud-Tabelle für Spielerprofile.
- Profilbilder oder individuelle Spielerfarben.
- Verwaltungsdialog für beliebige historische Alias-Zusammenführungen.
- Browser-End-to-End-Test-Suite; diese folgt in Paket E.
